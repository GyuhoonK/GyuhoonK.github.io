Bucket in Hive, Spark

HiveмҷҖ Sparkм—җм„ң Bucket м°Ёмқҙм җм—җ лҢҖн•ҳм—¬

Bucketting

BucketмқҖ мҝјлҰ¬ м„ұлҠҘ н–ҘмғҒмқ„ мң„н•ҙ мӮ¬мҡ©лҗ©лӢҲлӢӨ. нҠ№нһҲ Joinм—җм„ң мӮ¬мҡ©лҗҳлҠ” key columnмқ„ кё°мӨҖмңјлЎң bucketting н•  кІҪмҡ° нҒ° м„ұлҠҘ н–ҘмғҒмқ„ м–»мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

Bucketмқҳ кё°ліё м•„мқҙл””м–ҙлҠ” м§Җм •лҗң м№јлҹјмқ„ кё°мӨҖмңјлЎң hash functionмқ„ нҶөн•ҙ л§Ңл“Өм–ҙм§„ к°’м—җ н•ҙлӢ№н•ҳлҠ” bucket fileм—җ лҚ°мқҙн„°лҘј м ҖмһҘн•ҳлҠ” кІғмһ…лӢҲлӢӨ.

Bucket in Hive

HiveQLлЎң Partition, Bucketмқҙ м„ м–ёлҗң н…Ңмқҙлё”мқ„ м„ м–ён•ҳлҠ” мҝјлҰ¬лҠ” м•„лһҳмҷҖ к°ҷмҠөлӢҲлӢӨ.

CREATE EXTERNAL TABLE bucketed_tbl -- EXTERNAL TABLEлЎң мғқм„ұ
(name string, birthday timestamp) -- COLUMNмқ„ м„ м–ё, PARTITIONмңјлЎң мӮ¬мҡ©лҗҳлҠ” COLUMNмқҖ м ңмҷён•Ё
PARTITIONED BY (p_date string) -- PARTITION кё°мӨҖ м„ м–ё
CLUSTERED BY (name) -- HASH FUNCTIONм—җ мӮ¬мҡ©н•  кё°мӨҖ м„ м–ё
	SORTED BY (birthday) -- BUCKET лӮҙм—җм„ң м •л ¬ кё°мӨҖ м„ м–ё
	INTO 32 BUEKCTS -- BUCKET FILE к°ңмҲҳ м„ м–ё
STORED AS 'parquet'
LOCATION '/user/hive/warehouse/default.db/bucketed_tbl' -- EXTERNAL TABLEмқҳ м ҖмһҘ мң„м№ҳ
;

н•ҙлӢ№ н…Ңмқҙлё”м—җ INSERTлҠ” м•„лһҳмҷҖ к°ҷмқҖ мҝјлҰ¬лЎң мӢӨн–үн•©лӢҲлӢӨ(STATIC PARTITION).

INSERT INTO TABLE buckted_tbl PARTITION (p_date='20211010')
SELECT 
	name, birthday
FROM 20211010_source

INSERT мқҙнӣ„ LOCATIONм—җлҠ” м•„лһҳмІҳлҹј нҢҢмқјмқҙ м ҖмһҘлҗ©лӢҲлӢӨ.

Bucket in Spark(pyspark)

Sparkм—җм„ңлҸ„ partition, bucket кё°лҠҘмқ„ м ңкіөн•©лӢҲлӢӨ. мң„м—җм„ң мӢӨн–үн–ҲлҚҳ HiveQL мҝјлҰ¬лҠ” м•„лһҳ pyspark мҪ”л“ңмҷҖ к°ҷмқҖ н‘ңл©ҙм ҒмңјлЎңлҠ” к°ҷмқҖ мқҳлҜёлҘј м§ҖлӢҷлӢҲлӢӨ. мҰү HiveQLмқҙлӮҳ м•„лһҳ pyspark мҪ”л“ңлҠ” лӘЁл‘җ 1) External 2) Partition 3) Bucket мқ„ м„ м–ён•ҳкі  мһҲмҠөлӢҲлӢӨ.

source = spark.sql("SELECT * FROM 20211010_soucre")
source.write\
      .format('parquet')\
      .partitionBy('p_date')\
      .bucketBy(32, 'name')\
      .sortBy('birthday')\
      .mode('append')\
      .option('path', '/user/hive/warehouse/default.db/buckted_tbl')\
      .saveAsTable('default.bucketed_tbl')

32к°ңмқҳ bucket нҢҢмқјмқҙ мғқм„ұлҗ  кІғ к°ҷм§Җл§Ң, мқҙл ҮкІҢ м ҖмһҘн•ң кІҪмҡ°м—җ л§Өмҡ° л§ҺмқҖ parquet fileмқҙ мғқм„ұлҗңлӢӨлҠ” кІҒлӢҲлӢӨ.

# Bucket in Hiveм—җм„ң м ҖмһҘлҗң нҢҢмқјл“ӨмқҖ лӘЁл‘җ мӮӯм ңн–ҲлӢӨкі  к°Җм •н•ҳкі  мЎ°нҡҢ

# 12416к°ңмқҳ bucket file

мҷң мқҙл ҮкІҢ л§ҺмқҖ bucketл“Өмқҙ мғқм„ұлҗҳлҠ” кІғмқјк№Ңмҡ”?

мқҙлҠ” sparkм—җм„ң buckettingмқ„ н•  кІҪмҡ° к°Ғ MAP лі„лЎң bucekt fileмқ„ л§Ңл“Өкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

мң„ к·ёлҰјмІҳлҹј, Hiveм—җм„ңлҠ” Reducer н•ҳлӮҳк°Җ н•ҳлӮҳмқҳ bucketмқ„ л§Ңл“ңлҠ”лҚ° л№„н•ҙ, Sparkм—җм„ңлҠ” к°Ғ taskк°Җ мһҗмӢ мқҳ bucketмқ„ л§Ңл“Өкі  мқҙлҘј н…Ңмқҙлё” кІҪлЎң(path)м—җ м ҖмһҘн•©лӢҲлӢӨ.

кі„мӮ°н•ҙліҙл©ҙ \(32 * 388 = 12416\)мһ…лӢҲлӢӨ. мҰү, 388к°ңмқҳ taskк°Җ к°Ғк°Ғ bucket fileмқ„ мғқм„ұн•ҳмҳҖкё° л•Ңл¬ём—җ мң„мҷҖ к°ҷмқҙ 12416к°ңмқҳ bucket fileмқҙ м ҖмһҘлҗң кІғмһ…лӢҲлӢӨ. л”°лқјм„ң SparkлҘј нҶөн•ҙ buckettingмқ„ н•  л•ҢлҠ” мқҙлҹ¬н•ң л¶Җл¶„м—җ мЈјмқҳн•ҙм•јн•©лӢҲлӢӨ.

Hive Bucket vs. Spark Bucket

лҚ” нҒ° л¬ём ңлҠ” Hive Bucketкіј Spark Bucketмқҙ м–‘лҰҪн•  мҲҳ м—ҶлӢӨлҠ” кІғмһ…лӢҲлӢӨ.

SparkлҘј нҶөн•ҙ мғқм„ұлҗң Bucketed TableмқҖ Hiveм—җм„ң мЎ°нҡҢ л¶Ҳк°ҖлҠҘн•ҳлӢӨ

source = spark.sql("SELECT * FROM 20211010_soucre")
source.write\
      .format('parquet')\
      .partitionBy('p_date')\
      .bucketBy(32, 'name')\
      .sortBy('birthday')\
      .mode('append')\
      .option('path', '/user/hive/warehouse/default.db/buckted_tbl')\
      .saveAsTable('default.bucketed_tbl')

pysparkлЎң мң„мҷҖ к°ҷмқҙ м Ғмһ¬м—җ м„ұкіөн•ҳлҚ”лқјлҸ„, н…Ңмқҙлё”мқ„ мЎ°нҡҢн•ҳлҠ” мҝјлҰ¬лҘј мӢӨн–үн•  кІҪмҡ° м—җлҹ¬лҘј л°ңмғқмӢңнӮөлӢҲлӢӨ.

HiveQLлЎң мғқм„ұлҗң Bucketed Tableм—җлҠ” SparkлҘј нҶөн•ҙ INSERTн•  мҲҳ м—ҶлӢӨ

  1. лЁјм Җ Bucketed Tableмқ„ мғқм„ұн•ҳлҠ” м•„лһҳ мҝјлҰ¬лҘј мӢӨн–үн•©лӢҲлӢӨ.

    CREATE EXTERNAL TABLE bucketed_tbl -- EXTERNAL TABLEлЎң мғқм„ұ
(name string, birthday timestamp) -- COLUMNмқ„ м„ м–ё, PARTITIONмңјлЎң мӮ¬мҡ©лҗҳлҠ” COLUMNмқҖ м ңмҷён•Ё
PARTITIONED BY (p_date string) -- PARTITION кё°мӨҖ м„ м–ё
CLUSTERED BY (name) -- HASH FUNCTIONм—җ мӮ¬мҡ©н•  кё°мӨҖ м„ м–ё
	SORTED BY (birthday) -- BUCKET лӮҙм—җм„ң м •л ¬ кё°мӨҖ м„ м–ё
	INTO 32 BUEKCTS -- BUCKET FILE к°ңмҲҳ м„ м–ё
STORED AS 'parquet'
LOCATION '/user/hive/warehouse/default.db/bucketed_tbl' -- EXTERNAL TABLEмқҳ м ҖмһҘ мң„м№ҳ
;

  2. pysparkлҘј мқҙмҡ©н•ҳм—¬ Bucketed Tableм—җ лҚ°мқҙн„°лҘј append(INSERT)н•©лӢҲлӢӨ.

    м•„лһҳмҷҖ к°ҷмқҖ м—җлҹ¬ л©”мӢңм§ҖлҘј нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    The format of the existing table default.bucketed_tbl is 'HiveFormat'.
It doesn't match the specified format 'ParquetFileFormat'.

    HiveQLлЎң мғқм„ұн•ң Bucketed TableмқҖ HiveFormatмңјлЎң м ҖмһҘлҗҳм–ҙ мһҲм–ҙ ParquetFileFormat мңјлЎң нҢҢмқјмқ„ м Ғмһ¬н•  мҲҳ м—ҶлӢӨлҠ” л©”мӢңм§Җмһ…лӢҲлӢӨ.

  3. мқҙлІҲм—җлҠ” HiveFormatмңјлЎң м Ғмһ¬н•ҙліҙм•ҳмҠөлӢҲлӢӨ.

    source = spark.sql("SELECT * FROM 20211010_soucre")
source.write\
      .format('hive')\
      .partitionBy('p_date')\

    мқҙлІҲм—җлҠ” лӢӨлҘё м—җлҹ¬ л©”мӢңм§Җк°Җ нҷ•мқёлҗ©лӢҲлӢӨ.

    Creating buckted Hive Serde table is not supported yet.

    Sparkк°Җ м•„м§Ғ Bucketed Hive SerdeлҘј мғқм„ұн•ҳлҠ” кё°лҠҘмқ„ м§Җмӣҗн•ҳм§Җ м•Ҡкі  мһҲлӢӨлҠ” лӮҙмҡ©мһ…лӢҲлӢӨ. м ңк°Җ н…ҢмҠӨнҠём—җ мӮ¬мҡ©н–ҲлҚҳ spark лІ„м „мқҖ 2.3мһ…лӢҲлӢӨ.

    нҳ„мһ¬лҠ” 3.0 мқҙмғҒмқҳ лІ„м „мқҙ лҰҙлҰ¬мҰҲлҗң кІғмңјлЎң м•Ңкі  мһҲлҠ”лҚ° нҳ„мһ¬ мқҙ кё°лҠҘмқҙ кө¬нҳ„лҗҳм—ҲлҠ”м§ҖлҠ” нҷ•мқён•ҙліҙм•„м•јн•  кІғ к°ҷмҠөлӢҲлӢӨ.
